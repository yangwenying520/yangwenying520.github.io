新建文章

进入到hexo 安装目录，D:\Blog
在空白处右键，选择 Git Bash Here

在命令行输入 hexo new 新文章文件名，创建文件名为新文章文件名的新文章

编辑文章

修改 D:\Blog\source\_posts\新文章文件名.md，完成之后保存

发布新文章

确保当前目录为 hexo 安装目录，即D:\Blog

提交之后，等待内容刷新~
